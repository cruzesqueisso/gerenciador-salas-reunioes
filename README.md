## Gerenciador de salas de reunioes

### Stack utilizada:


 * Spring Web
 * Spring Data JPA
 * H2 Database
 * Java 8
 * Maven
 * Angular 
 * RxJS
 
### Endpoints criados na API

* Criar sala de reuniao
POST - /api/v1/rooms

* Listar todas as salas
GET - /api/v1/rooms

* Buscar uma sala pelo Id
GET - /api/v1/rooms/{id}

* Atualizar uma sala pelo Id
PUT - /api/v1/rooms/{id}

* Excluir uma sala pelo id
DELETE - /api/v1/rooms/{Id}
